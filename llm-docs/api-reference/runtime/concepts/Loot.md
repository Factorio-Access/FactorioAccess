# Loot

**Type:** Table

## Parameters

### count_max

Maximum amount of loot to drop.

**Type:** `double`

**Required:** Yes

### count_min

Minimum amount of loot to drop.

**Type:** `double`

**Required:** Yes

### item

Item prototype name of the result.

**Type:** `string`

**Required:** Yes

### probability

Probability that any loot at all will drop, as a number in range [0, 1].

**Type:** `double`

**Required:** Yes
