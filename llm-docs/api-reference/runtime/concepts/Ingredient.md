# Ingredient

**Type:** Table

## Parameters

### amount

Amount of the item or fluid.

**Type:** `double`

**Required:** Yes

### ignored_by_stats

How much of this ingredient is ignored by statistics.

**Type:** `uint` | `double`

**Optional:** Yes

### name

Prototype name of the required item or fluid.

**Type:** `string`

**Required:** Yes

### type

**Type:** `"item"` | `"fluid"`

**Required:** Yes
