# ModChangeData

**Type:** Table

## Parameters

### new_version

New version of the mod. `nil` if the mod is no longer present (i.e. it was just removed).

**Type:** `string`

**Optional:** Yes

### old_version

Old version of the mod. `nil` if the mod wasn't previously present (i.e. it was just added).

**Type:** `string`

**Optional:** Yes
