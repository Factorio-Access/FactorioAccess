# SelectedPrototypeData

**Type:** Table

## Parameters

### base_type

The base type of the prototype. E.g. `"entity"`.

**Type:** `IDType`

**Required:** Yes

### derived_type

The `type` of the prototype. E.g. `"tree"`.

**Type:** `string`

**Required:** Yes

### name

The `name` of the prototype. E.g. `"tree-05"`.

**Type:** `string`

**Required:** Yes

### quality

The `name` of the quality prototype if any. E.g. `"normal"`.

**Type:** `string`

**Optional:** Yes
