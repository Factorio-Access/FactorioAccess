# SignalID

**Type:** Table

## Parameters

### name

Name of the prototype.

**Type:** `string`

**Optional:** Yes

### quality

The prototype name of the signal's quality. Defaults to `normal`.

**Type:** `QualityID`

**Optional:** Yes

### type

The type of the signal. If the type is `"item"`, this will be `nil` when reading.

**Type:** `SignalIDType`

**Optional:** Yes
