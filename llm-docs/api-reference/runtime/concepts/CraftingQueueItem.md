# CraftingQueueItem

**Type:** Table

## Parameters

### count

The amount of items being crafted.

**Type:** `uint`

**Required:** Yes

### index

The index of the item in the crafting queue.

**Type:** `uint`

**Required:** Yes

### prerequisite

The item is a prerequisite for another item in the queue.

**Type:** `boolean`

**Required:** Yes

### recipe

The recipe being crafted.

**Type:** `string`

**Required:** Yes
