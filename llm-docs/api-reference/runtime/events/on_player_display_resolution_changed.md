# on_player_display_resolution_changed

Called when the display resolution changes for a given player.

## Event Data

### name

**Type:** `defines.events`

Identifier of the event

### old_resolution

**Type:** `DisplayResolution`

The old display resolution

### player_index

**Type:** `uint32`

The player

### tick

**Type:** `uint32`

Tick the event was generated.
