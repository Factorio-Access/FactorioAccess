# on_area_cloned

Called when an area of the map is cloned.

## Event Data

### clear_destination_decoratives

**Type:** `boolean`

### clear_destination_entities

**Type:** `boolean`

### clone_decoratives

**Type:** `boolean`

### clone_entities

**Type:** `boolean`

### clone_tiles

**Type:** `boolean`

### destination_area

**Type:** `BoundingBox`

### destination_force

**Type:** `LuaForce` *(optional)*

### destination_surface

**Type:** `LuaSurface`

### name

**Type:** `defines.events`

Identifier of the event

### source_area

**Type:** `BoundingBox`

### source_surface

**Type:** `LuaSurface`

### tick

**Type:** `uint32`

Tick the event was generated.
