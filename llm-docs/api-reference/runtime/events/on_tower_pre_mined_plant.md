# on_tower_pre_mined_plant

Called before an agricultural tower mines a plant.

## Event Data

### name

**Type:** `defines.events`

Identifier of the event

### plant

**Type:** `LuaEntity`

The plant that is about to be mined.

### tick

**Type:** `uint`

Tick the event was generated.

### tower

**Type:** `LuaEntity`

The tower that is about to do the mining.
