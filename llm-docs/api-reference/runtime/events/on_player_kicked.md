# on_player_kicked

Called when a player is kicked.

## Event Data

### by_player

**Type:** `uint32` *(optional)*

The player that did the kicking if any.

### name

**Type:** `defines.events`

Identifier of the event

### player_index

**Type:** `uint32`

The player kicked.

### reason

**Type:** `string` *(optional)*

The reason given if any.

### tick

**Type:** `uint32`

Tick the event was generated.
