# on_chart_tag_removed

Called just before a chart tag is deleted.

## Event Data

### force

**Type:** `LuaForce`

### name

**Type:** `defines.events`

Identifier of the event

### player_index

**Type:** `uint32` *(optional)*

### tag

**Type:** `LuaCustomChartTag`

### tick

**Type:** `uint32`

Tick the event was generated.
