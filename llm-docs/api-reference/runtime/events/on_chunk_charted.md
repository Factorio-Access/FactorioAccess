# on_chunk_charted

Called when a chunk is charted or re-charted.

## Event Data

### area

**Type:** `BoundingBox`

Area of the chunk.

### force

**Type:** `LuaForce`

### name

**Type:** `defines.events`

Identifier of the event

### position

**Type:** `ChunkPosition`

### surface_index

**Type:** `uint32`

### tick

**Type:** `uint32`

Tick the event was generated.
