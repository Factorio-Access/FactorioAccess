# on_robot_pre_mined

Called before a robot mines an entity.

## Event Data

### entity

**Type:** `LuaEntity`

The entity which is about to be mined.

### name

**Type:** `defines.events`

Identifier of the event

### robot

**Type:** `LuaEntity`

The robot that's about to do the mining.

### tick

**Type:** `uint`

Tick the event was generated.
