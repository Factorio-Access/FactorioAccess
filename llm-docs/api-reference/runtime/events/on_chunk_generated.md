# on_chunk_generated

Called when a chunk is generated.

## Event Data

### area

**Type:** `BoundingBox`

Area of the chunk.

### name

**Type:** `defines.events`

Identifier of the event

### position

**Type:** `ChunkPosition`

Position of the chunk.

### surface

**Type:** `LuaSurface`

The surface the chunk is on.

### tick

**Type:** `uint`

Tick the event was generated.
