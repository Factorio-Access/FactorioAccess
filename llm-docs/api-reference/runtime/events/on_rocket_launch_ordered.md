# on_rocket_launch_ordered

Called when a rocket silo is ordered to be launched.

## Event Data

### name

**Type:** `defines.events`

Identifier of the event

### player_index

**Type:** `uint` *(optional)*

The player that is riding the rocket, if any.

### rocket

**Type:** `LuaEntity`

### rocket_silo

**Type:** `LuaEntity`

### tick

**Type:** `uint`

Tick the event was generated.
