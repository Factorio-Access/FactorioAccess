# on_marked_for_deconstruction

Called when an entity is marked for deconstruction with the Deconstruction planner or via script.

## Event Data

### entity

**Type:** `LuaEntity`

### name

**Type:** `defines.events`

Identifier of the event

### player_index

**Type:** `uint32` *(optional)*

### tick

**Type:** `uint32`

Tick the event was generated.
