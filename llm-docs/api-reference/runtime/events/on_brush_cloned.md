# on_brush_cloned

Called when a set of positions on the map is cloned.

## Event Data

### clear_destination_decoratives

**Type:** `boolean`

### clear_destination_entities

**Type:** `boolean`

### clone_decoratives

**Type:** `boolean`

### clone_entities

**Type:** `boolean`

### clone_tiles

**Type:** `boolean`

### destination_force

**Type:** `LuaForce` *(optional)*

### destination_offset

**Type:** `TilePosition`

### destination_surface

**Type:** `LuaSurface`

### name

**Type:** `defines.events`

Identifier of the event

### source_offset

**Type:** `TilePosition`

### source_positions

**Type:** Array[`TilePosition`]

### source_surface

**Type:** `LuaSurface`

### tick

**Type:** `uint32`

Tick the event was generated.
