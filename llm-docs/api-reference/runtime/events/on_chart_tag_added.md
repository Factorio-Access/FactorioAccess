# on_chart_tag_added

Called when a chart tag is created.

## Event Data

### force

**Type:** `LuaForce`

### name

**Type:** `defines.events`

Identifier of the event

### player_index

**Type:** `uint` *(optional)*

### tag

**Type:** `LuaCustomChartTag`

### tick

**Type:** `uint`

Tick the event was generated.
