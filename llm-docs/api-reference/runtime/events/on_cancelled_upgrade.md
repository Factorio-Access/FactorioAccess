# on_cancelled_upgrade

Called when the upgrade of an entity is canceled.

## Event Data

### entity

**Type:** `LuaEntity`

### name

**Type:** `defines.events`

Identifier of the event

### player_index

**Type:** `uint32` *(optional)*

### quality

**Type:** `LuaQualityPrototype`

### target

**Type:** `LuaEntityPrototype`

### tick

**Type:** `uint32`

Tick the event was generated.
