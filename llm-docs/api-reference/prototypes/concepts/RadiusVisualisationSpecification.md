# RadiusVisualisationSpecification

Sprite to be shown around the entity when it is selected/held in the cursor.

**Type:** `Struct`

## Properties

*These properties apply when the value is a struct/table.*

### sprite

**Type:** `Sprite`

**Optional:** Yes

### distance

Must be greater than or equal to 0.

**Type:** `double`

**Optional:** Yes

**Default:** 0

### offset

**Type:** `Vector`

**Optional:** Yes

### draw_in_cursor

**Type:** `boolean`

**Optional:** Yes

**Default:** True

### draw_on_selection

**Type:** `boolean`

**Optional:** Yes

**Default:** True
